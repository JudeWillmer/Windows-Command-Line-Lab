# Windows Command Line Lab

## Objective

Practice and document essential Windows commands used by IT Support and Help Desk professionals.

## Topics Covered

### Network Troubleshooting
- ipconfig
- ipconfig /all
- ipconfig /release
- ipconfig /renew
- ipconfig /flushdns
- ping
- tracert
- nslookup

### User and System Management
- hostname
- whoami
- net user
- net localgroup
- systeminfo

### Group Policy
- gpupdate /force
- gpresult

### Shared Drive Troubleshooting
- net use

### System Diagnostics
- tasklist
- taskkill
- sfc /scannow

## Skills Demonstrated

- Network troubleshooting
- DNS troubleshooting
- User account management
- Group Policy troubleshooting
- Shared drive troubleshooting
- System diagnostics
